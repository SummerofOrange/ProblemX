# QuickStart

本部分将教你如何使用并快速上手ProblemX（v2.1版本）

## 1. 获取ProblemX

从[Release](https://github.com/SummerofOrange/ProblemX/releases)处下载最新的ProblemX程序并解压到一个你喜欢的地方。

## 2. 准备并导入题库

要使用ProblemX，你需要先有一个题库。在 v2.1 中，我们提供了多种获取题库的方式，并支持与辅助答题系统联动。

### 方式一：直接在软件中创建

1. 打开ProblemX程序，点击【配置程序】。
2. 点击左下角的【创建科目】，输入科目名称（如“高等数学”）。
3. 选中左侧刚创建的科目，点击右侧的【创建题库】，选择题型和模板，即可快速生成一个包含示例题目的题库。
4. 选中左侧题库列表中的题库项（勾选框启用），即可开始使用（v2.1新增左侧勾选框启用/禁用功能）。

### 方式二：从 PTA 智能获取（推荐）

1. 在【配置程序】界面选中科目，点击【自动获取题库】。
2. 在左侧浏览器登录 Pintia 账号，进入具体的题目集页面。
3. 点击下方的【解析当前页面的...】按钮，系统会自动抓取题目并识别评测状态（绿色=正确，红色=错误）。
4. 点击【保存当前题库】，勾选“保存时下载图片”，即可将题库保存到当前科目下（实现完全离线化）。

### 方式三：手动导入文件夹

1. 点击左下角的【添加科目】。
2. 选择包含题库文件的文件夹（需按 `科目/题型/*.json` 结构组织）。
3. 导入成功后即可在左侧看到科目列表（新导入题库将在点击“题目助手”时自动加载）。

### 方式四：使用脚本转换

对于雨课堂、学习通等平台，可以使用仓库提供的 Python 脚本将 HTML 转换为 JSON 题库，然后按方式三导入。

## 3. 开始练习

当你导入并配置好了至少一个题库后，你将可以开始练习（v2.1新增配置项）。

1. 回到主界面，点击【开始练习】。
2. 进行刷题即可（支持“乱序题目”开关控制是否洗牌，写入配置持久化）。

### 刷题

程序会自动读取你的题库，并根据“乱序题目”开关控制是否打乱呈现在页面上。

你将看到题目及其对应的选项，每次答完一题之后，你需要手动点击提交答案，程序会判断你答案的正确性，并提示“答案正确”或“答案错误”，并显示正确答案。完成上述步骤后，点击下一题即可。

### 题目列表

在右侧，你可以看到本次练习的所有题目，包括各个题目的作答情况。你可以点击对应的题目，进行题目间的跳转。

### 存档系统

当你一次选择了过多的题目时，你可以无法一口气完成整个练习。此时，点击右下角的【退出】，选择保存并退出，即可进行存档。在下次开始时，你的主页面将会出现【继续练习】的按钮，点击该按钮即可快速回到之前的存档。

### 错题系统

当你完成一次刷题时，系统将统计你所有错误的题目（不包括未作答的题）。你可以自由选择是否将他们导入到错题系统。

在主界面，点击【错题复习】，即可进入错题系统。你将在左侧看到所有的错题，并支持筛选。你可以自由标记其是否以及掌握。此外，你可以选中若干个错题，点击左下角的【复习错题】，此时将使用你选中的错题进行组卷并进入答题页面，进行错题的练习。

### 配置项操作（v2.1新增）
1. 题库详情的“抽取数量”支持“滑动条 + 数字输入框”双方式控制，并进行联动；
2. 新增“乱序题目”开关，支持控制练习时是否对题目进行洗牌，并写入配置持久化；
3. 左侧题库列表为每个题库项增加勾选框用于启用/禁用，交互更直观。

## 4. 辅助答题系统使用（v2.1新增）

1. 回到主界面，点击【题目助手】（系统将自动强制重建题库索引，新导入题库立刻生效）；
2. 左侧题目列表通过颜色标记状态（绿色-已作答，红色-新题/无匹配，黄色-未作答，蓝色-选中）；
3. 点击【自动答题】一键从本地题库搜索最佳匹配答案并填入网页（大幅提升练习效率）；
4. 点击【导出新题】将未匹配到的题目导出为JSON题库，便于扩充题库；
5. 查看日志面板，实时记录解析、搜索、填入等操作详情。

## 快捷键设置

为了方便答题，ProblemX支持许多快捷键。

- [A/B/C/D]：你可以在回答单选题时，使用键盘上的**[A/B/C/D]**进行快速选择答案。
- [T/F]：你可以在回答判断题时，使用键盘上的**[T/F]**进行快速选择答案(此外，同样也支持使用**[A/B]**选择答案)。

- [Enter]：你可以使用Enter快速提交答案。
- [Space/Shift+Space]：你可以使用空格/Shift+空格快速切换到下/上一题。
